Политика конфиденциальности

Дата вступления в силу: 28 марта 2026 г.

Расширение Insta Auditor Prompt Generator (далее — «Расширение») уважает конфиденциальность пользователей.

1. Какие данные обрабатываются

Расширение может обрабатывать только данные, необходимые для работы функций:

  • публичные данные Instagram-профиля, открытого пользователем;
  • введённые пользователем данные и действия в интерфейсе Расширения;
  • технические данные, сохраняемые локально в chrome.storage (например, настройки и состояние интерфейса).

2. Что мы не собираем

  • не собираем пароли от Instagram, Telegram и других сервисов;
  • не получаем доступ к приватным сообщениям;
  • не продаём и не передаём персональные данные третьим лицам в маркетинговых целях.

3. Куда отправляются данные

По действию пользователя данные (например, JSON профиля и сформированный промпт) могут отправляться:

  • на ваш сервер обработки;
  • в Telegram-бота, указанного владельцем сервиса.

Передача выполняется только для предоставления функции анализа.

4. Хранение данных

  • Локально: часть технических данных хранится в chrome.storage.
  • На сервере: данные могут временно храниться для обработки запроса и журналирования ошибок.

5. Безопасность

Мы применяем разумные технические меры для защиты данных при передаче и обработке. Однако ни один способ передачи данных через интернет не гарантирует абсолютную безопасность.

6. Права пользователя

Пользователь может:

  • удалить Расширение в любой момент;
  • очистить локальные данные браузера;
  • обратиться к владельцу сервиса по вопросам удаления или обработки данных.

7. Изменения политики

Мы можем обновлять эту Политику конфиденциальности. Актуальная версия публикуется по ссылке, указанной в карточке расширения.

8. Контакты
